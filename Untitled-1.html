The Image Classification is a web-based software, this is end to end data science and ML project on image classification. So, our end goal is to build a website where we can drag and drop an image of sports person and it will detect which sports person it is, in our web site, we restrict classification to only 5 people.

we are using google image scrapping tool to scrape images from google, and we have also used OpenCV which is an image processing library.

In terms of project architecture, we will build a model and python flask server for our website will make a call to that python flask server and we will get the response, mainly this project is classification project. The main purpose of this application is to classify the Images.

The Objective of this project is to classify the set of images. This application classifies the images efficiently but with average accuracy rate.

The goal of the companion project is to develop a real-time coordination system to dynamically create, maintain according to a decision-making mechanism, taking into historical and real-time information about the infrastructure. Our goal of the study is to evaluate different ML based approaches.

Tools And Technology used
  • Python
  • NumPy and OpenCV for data cleaning
  • Matplotlib and Seaborn for data visualization
  • Sklearn for model building
  • Jupiter notebook, visual studio code and PyCharm as IDE
  • Python flask for http server
  • HTML/CSS/JavaScript for UI
